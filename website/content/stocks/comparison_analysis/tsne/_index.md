```text
usage: tsne [-l LR] [-p] [-h]
```

Get similar companies to compare with using sklearn TSNE.

```
optional arguments:
  -l LR, --learnrate LR
                        TSNE Learning rate. Typical values are between 50 and 200 (default: 200)
  -p, --no_plot
  -h, --help            show this help message (default: False)
```

```
usage: news [-n N_NUM] [-h]
```

Prints latest news about company, including date, title and web link. [Source: News API]

```
optional arguments:
  -n N_NUM, --num N_NUM
                        Number of latest news being printed.
  -h, --help            show this help message
```

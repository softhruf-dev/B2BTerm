```
usage: rmv [-k RMV_STRIKE] [-h]
```

Remove one or all of the options from the diagram.

```
optional arguments:
  -k RMV_STRIKE, --strike RMV_STRIKE
                        remove option from diagram
  -a, --all             remove all options
  -h, --help            show this help message
```

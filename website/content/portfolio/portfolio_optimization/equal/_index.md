```
usage: equal [-v VALUE] [--pie] [-h]
```

Returns an equally weighted portfolio

```
optional arguments:
  -v VALUE, --value VALUE
                        Amount to allocate to portfolio (default: 1)
  --pie                 Display a pie chart for weights (default: False)
  -h, --help            show this help message (default: False)
```

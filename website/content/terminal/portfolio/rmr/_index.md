```
usage: rmr [-m MARKET] [-h]
```

A graph of portfolio returns versus a chosen benchmark or market.

```
optional arguments:
  -m MARKET, --market MARKET
                        Choose a ticker to be the market asset (default: SPY)
  -h, --help            show this help message (default: False)
```

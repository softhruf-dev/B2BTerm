```text
usage: acf [-l LAGS] [-h]
```

Auto-Correlation and Partial Auto-Correlation Functions for diff and diff diff stock data

```
optional arguments:
  -l LAGS, --lags LAGS  maximum lags to display in plots (default: 15)
```

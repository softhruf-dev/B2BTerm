```
usage: rise [-n N_NUM] [-h]
```

Print top rising related queries with this stock's query. [Source: Google]

```
optional arguments:
  -n N_NUM, --num N_NUM
                        number of top rising related queries to print. (default: 10)
  -h, --help            show this help message (default: False)
```

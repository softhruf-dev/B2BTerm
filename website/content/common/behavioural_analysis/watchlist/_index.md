```
usage: watchlist [-l N_LIMIT] [-h]
```

Searches [Reddit](https://reddit.com) for users' watchlists.

```
optional arguments:
  -l N_LIMIT, --limit N_LIMIT
                        limit of posts with watchlists retrieved. (default: 5)
  -h, --help            show this help message (default: False)
```

```
usage: regions [-n N_NUM] [-h]
```

Plot bars of regions based on stock's interest. [Source: Google]

```
optional arguments:
  -n N_NUM, --num N_NUM
                        number of regions to plot that show highest interest. (default: 10)
  -h, --help            show this help message (default: False)
```
